Descripción de Variables del Modelo

Esta sección detalla todas las variables utilizadas para construir y entrenar el modelo predictivo de riesgo de incendios. Se dividen en la variable objetivo que se busca predecir, las variables predictoras que alimentan al modelo, y las variables intermedias utilizadas durante el preprocesamiento de datos.

Variable Objetivo (Target)
Nombre Descripción Tipo Fuente de Datos Valores Posibles
`fire` Indica si en un punto y fecha determinados ocurrió un incendio. Binaria Positivos: FIRMS
Negativos: Muestreo aleatorio		 1 (Incendio)
0 (No Incendio)
Variables Predictoras (Features)
Categoría Nombre Descripción Tipo Fuente Unidad / Ejemplo
Topográficas elevacion Altura sobre el nivel del mar. Numérica DEM Metros (m)
pendiente Inclinación del terreno; influye en la velocidad de propagación. Numérica Derivado de DEM Grados (°)
orientacion_cat Dirección cardinal de la ladera (aspecto). Categórica Derivado de DEM N, NE, E, SE, etc.
Uso del Suelo cobertura Tipo de cobertura del suelo (bosque, pastizal, etc.). Categórica Clasificación Cobertura Códigos (3.0, 11.0)
Vegetación ndvi Índice de Vegetación de Diferencia Normalizada. Numérica MODIS (vía GEE) Rango: -1 a +1
Meteorológicas temperature Temperatura media diaria del aire. Numérica ERA5 (vía GEE) Grados Celsius (°C)
precipitation Precipitación total diaria. Numérica ERA5 (vía GEE) mm/día
wind_speed Velocidad media diaria del viento. Numérica ERA5 (vía GEE) m/s
humidity Humedad relativa media diaria del aire. Numérica ERA5 (vía GEE) Porcentaje (%)
Proximidad dist_vias Distancia a la vía principal más cercana. Numérica Shapefile de Vías Metros (m)
dist_ciudades Distancia al centro poblado más cercano. Numérica Shapefile de Ciudades Metros (m)
Variables Intermedias
Nombre Descripción Tipo Fuente Observaciones
geometry Coordenada (Punto) de cada observación. Geoespacial FIRMS / Muestreo Usada para extraer valores de los rasters.
date Fecha de cada observación. Fecha/Hora FIRMS / Muestreo Usada para enlazar con datos meteorológicos.
orientacion Orientación numérica en grados (0-360). Numérica DEM (calculado) Variable intermedia para crear `orientacion_cat`.
zonas_inflamables_poligono Polígono que define las áreas consideradas inflamables. Geoespacial Derivado de `cobertura` Usado para generar puntos de "No Incendio".
